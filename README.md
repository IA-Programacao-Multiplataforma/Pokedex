---

## 🚀 Como Executar o Projeto

### Clone este repositório

```bash
git clone https://github.com/IA-Programacao-Multiplataforma/Pokedex.git
```

### Acesse a pasta do projeto

```bash
cd Pokedex
```

### Instale as dependências

```bash
npm install
```

### Execute a aplicação

```bash
npx expo start
```

Utilize o aplicativo **Expo Go** em seu dispositivo móvel ou um emulador Android/iOS para visualizar a aplicação.
